- ✅ Thêm, sửa, xóa công việc

## 🛠️ Cài đặt và chạy local

### 📋 Yêu cầu:
- Node.js >= 18
- pnpm / yarn / npm

### 🚀 Các bước thực hiện:

```bash
# 1. Clone repository
git clone https://github.com/MonGen2211/to-do-app-NextJS.git
cd to-do-app-NextJS

# 2. Cài đặt dependencies
pnpm install
# Hoặc nếu dùng yarn
# yarn
# Hoặc npm
# npm install

# 3. Chạy dev server
pnpm dev
# Truy cập http://localhost:3000 trên trình duyệt
